Let's start mining in 3 easy steps!

  •  Choose your mining software depending on your graphics card (AMD or Nvidia)
  •  Download and install a wallet for your coins
  •  Create a launcher file that starts your miner and connects to our pool
    • Step 1: Download mining software to your computer.

    HashThat.net supports crypto currencies you can mine with the GPU in your computer. We are currently supporting Equihash coins, such as Bitcoin Private and Zclassic.


    Click the buttons below to download mining software depending on if you have an AMD or Nvidia card.

    AMD Mining Software:


    Claymores

    nVidia Mining Software:


    EWBF   DSTM

    Once you've selected your miner of choice..

    1. Download and extract (unzip) your chosen mining software to its own folder e.g. c:\mining
    2. Leave this folder window open, we'll come back to it in step 3.


    Step 2: Get a wallet
    Bitcoin Private Wallets:

  • Full Wallet (Recommended, helps strengthen the network).
  • Electrum Wallet

  • Electrum Wallet

    Step 3: Create a simple launcher file
  • Create a new txt file in the folder we made in step 1 (which should contain the mining software you unzipped/copied there).
  • Right click in the folder, select New -> Text Document, rename "New Text Document.txt" to hashthat.bat.
  • If you can't see the file extensions (.txt) you will need to enable them. Guide HERE
  • Copy and paste the example code below (relevant to the miner you chose) into your text file ("hashthat.bat") and save your file
  • Do not forget to insert your OWN wallet address and remove the wallet address from the example.
  • You should now be able to start mining by double clicking hashthat.bat to start the miner.

    • Here's the example code for each miner:

    

Claymores (AMD):


BTCP example:
ZecMiner64.exe -zpool stratum+tcp://stratum.hashthat.net:3072 -zwal b1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg -zpsw x

---------------------------------------------------------------------------------------------------------------


DSTM Miner (Nvidia):


BTCP example:
zm --server stratum.hashthat.net --port 3072 --user b1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg --pass x

---------------------------------------------------------------------------------------------------------------


EWBF Miner (Nvidia):

BTCP example:
miner --server stratum.hashthat.net --port 3072 --user b1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg --pass x
