Let's get started!

HashThat.net supports crypto currencies you can mine with the GPU in your computer.


The process is actually very easy, and we will do it in 3 simple steps:

  •  Choose your mining software depending on your graphics card (AMD or Nvidia)
  •  Download and install a wallet for your coins
  •  Create a launcher file that starts your miner and connects to our pool


Recommended Software for mining

AMD Mining Software:


Claymores

nVidia Mining Software:


EWBF   DSTM

General Windows Guide

  1. Download and extract (unzip) your chosen mining software to its own folder
  2. Create a new txt file in the new folder (which should contain the mining software) named "hashthat.bat"
    • Right click in the folder, select New -> Text Document, rename "New Text Document.txt" to hashthat.bat.
    • If you can't see the file extensions (.txt) you will need to enable them. Guide HERE
  3. Use our command line creator (below) to generate the necessary commands to suit our pool with your details
  4. Copy and paste the generated batch file into your text file ("hashthat.bat") and save your file
  5. You should now be able to start mining by double clicking hashthat.bat to start the miner.


  • See below further examples of batch files for mining with hashthat.net:

// *************
// Claymores
// *************
//
// ZecMiner64.exe -zpool POOL_ADDRESS:PORT -zwal WALLET_ADDRESS -zpsw x
//
// ZClassic example:
ZecMiner64.exe -zpool stratum+tcp://stratum.hashthat.net:3032 -zwal t1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg -zpsw x
//


// *************
// DSTM Miner
// *************
// 
// zm --server POOL_ADDRESS --port PORT --user WALLET_ADDRESS --pass x
//
// ZClassic example:
zm --server stratum.hashthat.net --port 3032 --user t1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg --pass x
//


// *************
// EWBF Miner
// *************
// 
// miner --server POOL_ADDRESS --port PORT --user WALLET_ADDRESS --pass x
//
// ZClassic example:
miner --server stratum.hashthat.net --port 3032 --user t1Snt2aUiqyL7B2tCBaBGeoCSJXQEuYhbkg --pass x
//
